# Unity Project: Basic Movement

## Описание
# Имена

Они не соответствуют обычному стилю C# и Unity.

Проблемные имена:

- GoPlaces — непонятное название. 
- NumberOfPlaceInArrayPlaces — слишком длинно и тяжело читается
- AllPlacespoint — грамматически странное имя
- arrayPlaces — префикс array в имени не нужен
- _float — плохое имя, по нему неясно, что это за значение
- abcd — технический мусор вместо имени переменной
- _pointByNumberInArray — слишком длинно и с лишними деталями реализации
- NextPlaceTakerLogic() — название неестественное

#  Поля
- поля объявлены хаотично.

## Используемые технологии
- Unity 6.3
- C#

## Структура

## corection

# По стилю

- нормальные имена
- приватные поля через _camelCase
- SerializeField вместо лишнего public
- методы private
- нет мусорных имен вроде abcd

# По логике

- есть защита от пустых ссылок
- нет лишнего GetComponent<Transform>()
- нет опасного сравнения позиций через ==
- метод выбора следующей точки делает только одно действие

## Как запустить
1. Открыть проект в Unity Hub
2. Загрузить сцену Main
3. Нажать Play
